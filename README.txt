#chem_mech_properties
# Material Science & Element Properties Applications
#Metallurgical Properties Calculator
This repository contains two Streamlit applications for material science and chemical element analysis.

## Applications Overview

### 1. Material Properties Analyzer (streamlit_mech_properties.py)
A comprehensive tool for analyzing and comparing thermal, electrical, and mechanical properties of various materials.

#### Features:
- *Thermal Properties Analysis*
  - Thermal conductivity
  - Specific heat capacity
  - Thermal expansion coefficient
  - Melting points

- *Electrical Properties Analysis*
  - Electrical conductivity
  - Resistivity
  - Dielectric constant

- *Mechanical Properties Analysis*
  - Tensile strength comparison
  - Yield strength
  - Young's modulus
  - Hardness measurements
  - Ductility properties

- *Material Comparison Tools*
  - Side-by-side material property comparison
  - Interactive charts and visualizations
  - Material selection guidance

#### Supported Materials:
- Metals (Steel, Aluminum, Copper, Titanium, etc.)
- Polymers (Nylon, PVC, PTFE, etc.)
- Ceramics (Alumina, Silicon Carbide, etc.)
- Composites (Carbon Fiber, Fiberglass, etc.)

### 2. Element Query Bot (streamlit_element_querybot_.py)
An interactive application for exploring chemical element properties from the periodic table.

#### Features:
- *Element Search & Discovery*
  - Search by element name, symbol, or atomic number
  - Browse complete periodic table
  - Filter elements by category (metals, non-metals, noble gases, etc.)

- *Comprehensive Property Data*
  - Basic properties (atomic number, weight, symbol, group, period)
  - Physical properties (density, melting point, boiling point, phase)
  - Electronic configuration
  - Atomic radius and electronegativity
  - Discovery information

- *Interactive Visualizations*
  - Property trends across periods and groups
  - Comparative analysis tools
  - Customizable data views
